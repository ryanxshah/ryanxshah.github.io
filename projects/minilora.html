Project

miniLoRA

A LoRA fine-tuning framework built from scratch in PyTorch. Makes it possible to adapt large pre-trained models to new tasks by training only a small fraction of the original parameters.

↗ GitHub Repo

Overview

miniLoRA implements Low-Rank Adaptation (LoRA), a parameter-efficient fine-tuning technique that injects trainable low-rank matrices into the weight layers of a frozen pre-trained model. Rather than updating all of the original weights during fine-tuning, only the small adapter matrices are trained - dramatically reducing memory usage and compute while preserving the knowledge already encoded in the base model.

How it works

For each target weight matrix in the model, LoRA adds two small trainable matrices - one that projects down to a low rank and one that projects back up. During fine-tuning, only these adapter matrices are updated while the original weights stay frozen. At inference time, the adapter outputs are added directly to the frozen layer outputs, so no extra latency is introduced.
