Project

Autonomous Car

a self-driving car model built on a transformer-based vision architecture. trained to predict steering decisions directly from camera input.

↗ GitHub Repo

Overview

this project applies transformer-based vision models to the autonomous driving problem. rather than relying on hand-crafted rules or sensor fusion pipelines, the model learns to map raw image frames directly to driving decisions — treating steering as a sequence prediction task over visual tokens.

How it works

input frames are tokenized and passed through a vision transformer that extracts spatial features from the image. these features are used to predict the steering angle at each timestep. the model is trained on recorded driving data using supervised learning, minimizing the difference between predicted and actual steering values.
